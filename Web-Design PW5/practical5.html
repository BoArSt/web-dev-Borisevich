Questions

A. What is a box model in CSS. Give box-sizing property definition.

The box model in CSS defines how elements are structured with content, padding, borders, and margins. The box-sizing property determines how the total width and height of an element is calculated, including or excluding padding and border.

B. What is a “clearfix hack” and when does it need to be applied.

The "clearfix hack" is a technique used to fix issues with floated elements not properly expanding the parent container's height. It needs to be applied when you have floated elements inside a container and the container collapses or doesn't stretch to contain all the floated elements.

C. What is a “margin collapse” and what are the use cases for margin: auto and negative margin values.

Margin collapse happens when the top and bottom margins of two adjacent elements are combined into a single margin. The margin: auto property is used to center an element horizontally within its container, while negative margin values can be used to create overlapping effects or adjust spacing between elements.
